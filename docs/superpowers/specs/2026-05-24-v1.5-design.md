# v1.5 Design Spec — 浏览页回归默认首页

> 日期：2026-05-24 | 状态：已批准

## 概述

将照片浏览恢复为默认首页，相册选择改为二级页面。3 列 × 2 行缩略图。

## 路由

```
Tab:  index.tsx → 照片浏览（默认首页）
Tab:  hub.tsx → 柱状图
Tab:  settings.tsx → 个人中心
非Tab: albums.tsx → 相册选择（push from index）
非Tab: review.tsx → 确认删除
非Tab: paywall.tsx → 会员

删除: browse.tsx
```

## 数据流

- `PhotoContext` 新增 `selectedAlbum: {id,title} | null` + `setSelectedAlbum`
- index.tsx 左上角相册按钮 → push /albums
- albums.tsx 选择相册 → setSelectedAlbum → router.back()
- index.tsx useEffect 监听 selectedAlbum 变化 → reloadPhotos
- review.tsx 用 context 的 selectedAlbum 导航回浏览页

## 相册页 UI

- 3 列 grid，每格正方形缩略图
- FlatList 分页，每页 6 张（2 行）
- 尺寸: `(screenW - 32 - 24) / 3`

## 改动文件

| 文件 | 操作 |
|------|------|
| app/index.tsx | 重写为浏览 + 相册按钮 |
| app/albums.tsx | 新建（相册选择页） |
| app/browse.tsx | 删除 |
| app/_layout.tsx | 路由调整 |
| app/review.tsx | 导航适配 |
| src/contexts/PhotoContext.tsx | 新增 selectedAlbum |
